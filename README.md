## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de corrigir problemas de acessibilidade, semântica HTML5 e SEO presentes no site original inspirado no projeto Inaccessible Gallery.

A proposta da atividade era transformar um site com falhas estruturais e de acessibilidade em uma aplicação moderna, organizada, responsiva e acessível para diferentes tipos de usuários.

Além das correções obrigatórias solicitadas na atividade, foram implementadas melhorias extras para tornar a experiência mais profissional e inclusiva.

# 📚 RELATÓRIO DO PROJETO

## Introdução

A acessibilidade web é essencial para garantir que todas as pessoas consigam utilizar sistemas digitais de maneira adequada.

Neste projeto foi realizada a reformulação de um site originalmente com diversos problemas de acessibilidade e semântica.

O objetivo foi aplicar conceitos modernos de desenvolvimento front-end utilizando HTML5 semântico, acessibilidade básica baseada em WCAG e boas práticas de SEO.

---

# 🔍 Análise do Site Original

O site original apresentava diversos problemas:

## Problemas Estruturais

* Falta de organização semântica
* Uso excessivo de `div`
* Estrutura pouco organizada

## Problemas de Acessibilidade

* Falta de contraste
* Falta de acessibilidade visual
* Navegação limitada
* Ausência de recursos assistivos modernos

## Problemas de SEO

* Estrutura de títulos limitada
* Pouca organização semântica

---

# 🛠️ Melhorias Aplicadas

## Estrutura HTML5

Foram utilizadas tags semânticas para melhorar:

* Organização
* SEO
* Compatibilidade com leitores de tela

---

## Acessibilidade

Foram adicionados:

* Labels
* aria-label
* Melhor contraste
* Leitor de voz
* Navegação acessível

---

## Interface Visual

Foi criada uma interface moderna utilizando:

* Glassmorphism
* Dark mode
* Light mode
* Paleta de cores moderna
* Layout responsivo

---

## Sistema Multilíngue

Foi desenvolvido um sistema de troca dinâmica entre:

* Português
* Inglês

Melhorando acessibilidade internacional e SEO.

---

# ⭐ Funcionalidades Extras

Além do solicitado na atividade, também foram adicionados:

* Sistema de leitura em voz alta
* Alternador de idiomas
* Modo claro/escuro
* Melhor experiência visual

---

# ✅ Conclusão

O projeto final conseguiu transformar um site com problemas estruturais e de acessibilidade em uma aplicação moderna, organizada e acessível.

As melhorias implementadas proporcionaram:

* Melhor experiência do usuário
* Melhor acessibilidade
* Melhor SEO
* Interface moderna
* Compatibilidade com tecnologias assistivas

Além disso, o projeto demonstra a importância da acessibilidade no desenvolvimento web moderno.
