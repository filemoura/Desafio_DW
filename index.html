Projeto de acessibilidade web

Quase todos os sites modernos possuem problemas de acessibilidade.

Inspirado no projeto original Inaccessible Gallery, este site demonstra os principais problemas encontrados na web e como pequenas melhorias podem transformar completamente a experiência do usuário.

Sobre o projeto

Visualizando barreiras digitais de forma moderna e acessível.

Esta versão foi totalmente reformulada utilizando HTML5 semântico, acessibilidade moderna, boas práticas de SEO e uma interface visual profissional.

O objetivo é demonstrar como pequenos erros de desenvolvimento podem impactar diretamente milhões de usuários.
